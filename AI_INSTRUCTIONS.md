# 智能菜谱系统 (Intelligent Recipe System) - 用户手机端 (H5) 开发指南

## 1. 项目概述
本项目是“智能菜谱系统”的面向普通用户的手机端应用（H5/Web App）。
【当前开发阶段声明】：目前处于纯前端页面开发和交互还原阶段，暂不对接真实后端，数据采用前端 Mock 或写死的 JSON 数据。

## 2. 技术栈约束与规范
在生成、修改代码时，请严格遵守以下技术栈：
- **核心框架:** Vue 3 (强制使用 Composition API 和 `<script setup>` 语法)
- **构建工具:** Vite
- **UI 组件库:** NutUI 4.x (京东移动端组件库，优先使用其内置组件还原设计)
- **CSS 方案:** Tailwind CSS (仅用于辅助 NutUI 无法覆盖的特殊布局、间距和基础排版)
- **路由管理:** Vue Router 4
- **网络请求:** Axios (统一放置在 `src/api` 目录下)
- **图标使用:** 优先使用 NutUI 内置的 Icon 图标，也可以利用unplugin-icons插件引入其他图标库（如mdi等）。
